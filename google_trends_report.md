## 7. Methodology Notes

- **API Access:** Google Trends API returned 400/429 errors for 7-day timeframes. Data gathered using monthly (1-m) and longer (3-m, 12-m) timeframes.
- **Search Interest Scale:** Values are relative (0-100) representing search interest relative to the highest point on the chart for the given region and time.
- **Daily Trending Searches:** Retrieved from the Google Trends RSS feed (trends.google.com/trending/rss?geo=US).
- **Rate Limiting:** Multiple requests were rate-limited (HTTP 429). Data collected across multiple sessions with cooldown periods.
- **Library Used:** pytrends 4.9.2 for Python API access.
